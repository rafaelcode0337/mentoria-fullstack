listas

podemos representar listas em nosso site com duas tags diferentes, <ul> e <ol>, e cada elemento dentro da lista é representado por um <li>

Listas não ordenadas

<ul> respresenta uma lista não ordenada, onde a ordem dos elementos não importam, exemplo uma lista de compras:

  • arroz
  • feijão
  • oléo
  • carne
  • ovos
  • leite

Listas ordenadas

<ol> respresenta uma lista ordenada, onde a ordem dos elementos importam, exemplo uma receita de doce de leite:

  1. jogue o leite e o açucar na penela
  2. mexa até ganhar consistência
  3. deixe na geladeira até ficar dura

Âncoras

Âncoras não são nada mais nada menos do que links, e existem ancôras internas que apontam para páginas do nosso site, ou elementos dentro dela, para definir uma âncora usamos a tag <a;a>, e para definirmos a url a que ela leva usamos o atributo href
